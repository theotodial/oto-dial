# Assets Folder

## Logo

Place your logo image file here:
- **File name:** `otodial-logo.png`
- **Location:** `frontend/src/assets/otodial-logo.png`
- **Usage:** Import in components using: `import logo from '../assets/otodial-logo.png'`

The logo should be:
- PNG format
- Transparent background (recommended)
- High resolution for crisp display
